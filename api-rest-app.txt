
Pre-requisites

1- Create 2 JSON files with authors and books 
2- Author have id, name and lastname
3- Book have id, name and authorId

Create a RestApi with the followinf endpoints

1- Get all authors
2- Get all books with the author
3- Add an author
4- Add a book
5- Modify an author
6- Modify a book
7- Delete a author
8- Delete a book
